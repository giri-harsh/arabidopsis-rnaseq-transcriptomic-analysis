# Dataset

## Bundled Example Dataset (Synthetic)

`data/example_dataset/` ships with the repository. It is generated by
`scripts/generate_example_dataset.py` using a Negative-Binomial simulation
model with a seeded ground-truth set of differentially expressed genes.

**This is not real biological data.** It exists so the pipeline can be
cloned and run immediately, with no external download and no licensing
concerns. See `data/example_dataset/README.md` for exact generation
parameters.

## Real Dataset (for actual biological analysis)

To run a genuine biological analysis, supply a public *Arabidopsis
thaliana* abiotic-stress RNA-seq count dataset. Suitable candidates are
raw-count RNA-seq series on NCBI GEO / SRA studying drought, salinity,
heat, or cold stress in *Arabidopsis thaliana*, with:

- A raw (non-normalized) gene × sample count matrix.
- Sample metadata identifying control vs. stress conditions and replicate
  structure.

### How to find one

1. Search NCBI GEO (https://www.ncbi.nlm.nih.gov/geo/) for
   `Arabidopsis thaliana abiotic stress RNA-seq raw counts`.
2. Prefer series (GSE accessions) that provide a supplementary raw count
   matrix file directly (many RNA-seq submissions include one), rather
   than only raw FASTQ — this pipeline starts from an already-quantified
   count matrix and does not perform alignment/quantification.
3. Note the accession number, license terms, and citation given on the
   GEO record page.

### Download instructions (example pattern)

GEO supplementary files can typically be fetched directly:

```bash
# Replace GSExxxxx and the file name with the actual accession/file
wget "https://ftp.ncbi.nlm.nih.gov/geo/series/GSExxxnnn/GSExxxxx/suppl/GSExxxxx_raw_counts.csv.gz" \
    -O data/raw/counts.csv.gz
gunzip data/raw/counts.csv.gz
```

Construct the matching `data/raw/metadata.csv` by hand from the GEO
series' sample characteristics table (sample id, condition, replicate).

### Licensing

Raw data from GEO/SRA is **not redistributed in this repository**. Users
must download data directly from the original source and comply with
the license/terms stated on that GEO/SRA record (GEO data is generally
freely available for research use, but always confirm the specific
series' terms and cite the original publication).

### Expected file format

**Counts** (`--counts`): CSV/TSV, first column = gene id, remaining
columns = one per sample, raw (non-normalized) integer counts.

```
gene_id,control_1,control_2,stress_1,stress_2
AT1G01010,120,98,340,410
AT1G01020,45,52,40,38
```

**Metadata** (`--metadata`): CSV/TSV, one row per sample.

```
sample_id,condition,replicate
control_1,control,1
control_2,control,2
stress_1,stress,1
stress_2,stress,2
```
