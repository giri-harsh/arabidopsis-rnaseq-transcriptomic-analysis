# Example Dataset (SYNTHETIC)

This dataset is generated by `scripts/generate_example_dataset.py` using a Negative-Binomial simulation model and is **not** real Arabidopsis thaliana sequencing data.

It exists so the pipeline can be run and tested immediately after cloning the repository, without downloading external data.

- Genes: 2000
- Samples: 4 control, 4 stress
- Seeded true DEGs: 200 (10% of genes, split up/down-regulated)
- Random seed: 42

For the real dataset used in the published analysis, see `docs/DATASET.md` at the repository root.
